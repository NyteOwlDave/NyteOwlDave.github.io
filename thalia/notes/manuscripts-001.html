[?] Link Table Creator

This is a manuscript for Q-Editor. It creates checklists from Navigation Docs. It also provides a bunch of list and text manipulation functions.

One of the more powerful features is the ability to scrape comments with special Marker tags (prefixes). This is very similar to the Cody self-documenting paradigm.
