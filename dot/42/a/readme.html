Dot Rocket Generator

This workspace is intended to provide the ability to generate a group of empty Dot Rocket files from a template file.

The template file is name template.html.

The Bash script that carries out this task is named make-dots.

The workspace is located in the System 42 dots folder. Access can be gained from most workspaces and hosts on the LAN. There are also access routes from the Web now. The most commonly available is via the Optimum IP4 address 173.213.25.64, port number 64264. This port is monitored on the Omega workstation, with the R6700 router providing IP Port Forwarding.

The port listener is an instance of the Python HTTP Server, which can be started using the file-server script located in the workspace dedicated for that purpose.
