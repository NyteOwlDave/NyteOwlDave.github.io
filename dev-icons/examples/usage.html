Using Design Icons

The easiest way to embed one of these Icons is to copy the raw SVG code from the file and paste that code into the HTML document.

Though this technique avoids additional HTTP fetch operations, it also causes the HTML document to become ugly and bloated.

There's a much simpler approach, fortunately. This happens to be the preferred technique.

The source code for this example HTML document contains both approaches.

If you prefer, you can example the source code directly using the browser or some download tool like curl or wget.

However, for simplicity, I've included the preferred approach as part of this document's visible content.
