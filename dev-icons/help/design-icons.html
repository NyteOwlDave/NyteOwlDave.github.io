Design Icons Help

Options

There are three numeric input fields:

  1. Index
  2. Count
  3. Limit

Index Option

This input determines the index of the first icon to be shown is the display preview area.

The range is clamped so that is falls within a lower limit of zero and an upper limit defined by the limit input.

You can change this value manually. It's also updated for you when you use the media buttons for scrolling.

Count Option

This input determines the number of icons to be displayed at any given location within the index range.

We use is a sliding window of indices. This value determines the size of that window.

Limit Option

This input is meant primarily for debugging and development.

However, it's also useful if the number of icon files is altered for some reason. Perhaps new icons were updated without reflecting this change in the application logic.
