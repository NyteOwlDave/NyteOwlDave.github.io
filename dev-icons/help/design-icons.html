Help Manual

Page Layout

The page typically appears similar to the figure below.

Navigation Menu

This menu is a collection of anchor ( A ) elements.

It's located in the HEADER element pinned along the top of the page.

Preview Area

The largest area is reserved for previewing icons.

The area is sandwiched between the Navigation Menu ( HEADER ) and the Options Section ( FOOTER ).

Options Section

This section is dedicated to managing input values.

It's located within the FOOTER element pinned at the bottom of the page, above the Action Menu.

As shown above, there are three numeric input fields:

  • Index
  • Count
  • Limit

NOTE: The labels are shown here for clarity. They're not displayed on the actual page.

Index Option

This input determines the Index of the first icon to be shown within the Preview Area.

The Range is clamped so that it falls within a lower limit of 1 and an upper limit defined by the Limit input.

You can change this value manually. It's also updated for you when you use the Media Buttons for scrolling.

Count Option

This input determines the Number of icons to be displayed at any given location within the index Range.

We use a Sliding Window of indices. This value represents the Size of that window.

Limit Option

This input is meant primarily for debugging and development.

However, it's also useful if the number of icon files is altered for some reason. Perhaps new icons were uploaded without reflecting this change in the Application's logic.

Action Menu

This section is dedicated to managing command actions.

It's located within the FOOTER element pinned at the bottom of the page, below the Options Section.

Update Button

The Update button reads all input values and performs validation.

These new values are used to modify the icons shown in the Preview Area.

Reset Button

The Reset button restores all input values to their original states.

The Preview Area is updated accordingly.

Media Buttons

These should be familiar, as they're used for most Media Control applications.

Icon Preview

If you'd like to preview an icon in a popup window, you just left click on that icon.

Note that modern browsers often prevent popup windows by default. You may receive a warning message (or just no popup). In the former case, the browser will probably offer you the opportunity to enable popups. Do so and try again, if you so chose.

If there is no warning message, you may need to open your browser's settings and search for security and/or permission options. There should be a way to permit popups via browser settings.

If all else fails, ask your favorite AI Chat Bot how to enable popups for your web browser. We'd suggest   Gemini for this. Since Google makes both Gemini and the Chrome browser.

Embedding Source Code

If you're designing a web page and want to use these icons, it's really easy. Just add some HTML to your project.

This page can generate some example code for you. Just hold down the SHIFT key while left-clicking any icon.

You should see a Popover Window with code you can copy and paste into your page's HTML source code.

NOTE : Most modern browser support Popovers. For those older browsers, you'll see the popover at all times. Clicking an icon will merely update the code it displays.
